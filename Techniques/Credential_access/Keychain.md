# Keychain

MITRE ATT&CK technique [T1142](https://attack.mitre.org/wiki/Technique/T1142)

Tactic: Credential Access

Platform: Mac

### Deception Techniques
* Add URL honeytokens and fake credentials (i.e. honey accounts) to the Keychain
